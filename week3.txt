Control Flow
1.Statements and Blocks
    - expression กลายเป็น statement เมื่อมีเครื่องหมาย ; ปิดท้าย
    - ฟังก์ชัน, if, else, while, for จะใช้ block ในการรวม statement หลายอันเข้าด้วยกัน
    - Null statement คือ statement ที่ปราศจากส่วนของ expression โดยเหลือเพียง ; เท่านั้น 
2.If-Else
    if-else จะต้องอยู่ต่อจากกันถึงจะทำให้เข้าเงื่อนไข else
    ตัวอย่างเช่น if 12>x
              if 12<x
              else
      else จะทำงานก็ต่อเมื่อเข้าเงื่อนไข if ที่สอง
3.Else-If
    else if จะทำงานได้ก็่ต่อเมื่อ เงื่อนไข if แรกเป็นเท็จ
    ถ้าเงื่อนไข if เป็นเท็จก็จะทำเรื่อยๆจนกว่าจะเจอเจออันที่เป็นจริง   
4.Switch 
    -ใช้ในการตรวจสอบเงื่อไขลักษณะที่ดูว่าค่าของ expression นั้น match เข้ากับค่าคงที่ตัวใด หลังจากนั้นโปรเเกรมจะกระโดดมาทำงาน statement 
    ของค่าจำนวนต็มนั้นลงมาเรื่อยๆ จนกว่าจะเจอ break จึงจะหลุด swticth
    -default เป็น optional มีหรือไม่มีก็ได้ เเต่ถ้ามีเเล้ว คำสั่ง statement ที่อยู่ภายใต้ default จะถูกทำงานเมื่อค่าของ expression ไม่ match
    กับ case ใดเลย
5.Loops - While และ For
    -ถ้า expression เป็นจริง statement จะถูกทำงาน เเละจะทำงานไปเรื่อยๆจนกว่าจะเป็นเท็จ
    -for(;;) เป็น infinite loob         
6.Loops - Do-While
    -ลูปนี้จะวนทำซ้ำไปเรื่อยๆ จนกว่า expression จะเป็นเท็จ ซึ่งถ้ามีค่าเป็นจริง statement จะถูกทำงานอีก เเละทำไปเรื่อยๆจนกว่าจะเป้นเท็จ
7.Break เเละ Continue   
    -ช่วยควบคุมการวนซ้ำได้มากขึ้น
    -Break ทำหน้าที่ให้ดปรเเกรมหลุดออกมาจากลูป
    -Continue จะบังคับการทำงานของลุปให้เกิดการวนซ้ำขึ้นมาใหม่
